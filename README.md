# full-graduation-project
This is a full-stack web application built with:

•Frontend: React.js  

•Backend: Node.js + Express  

•Database: MongoDB (or your DB)

# Folder Structure

•/frontend – React client
•/backend – Node.js API

# Getting started

 #Frontend
cd frontend
npm install
npm start

 #Backend
cd backend
npm install
node server.js
